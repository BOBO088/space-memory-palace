> **核心目标：找到第一个愿意付 99-999 元的人。** 不是更多代码。

## 0. 你 24 小时内要做的 4 件事

1. **打开 4 个示例链接**（任何人能访问），录 30s + 3min 两个版本的视频
2. **找 10 个人发链接**，按"会用 / 愿付 / 付多少"问，不问"觉得怎样"
3. **整理 3 维度统计**：最爱 / 最困惑 / 最愿付——决定下一个迭代方向
4. **绑定一个自己的域名**（可选；vercel.app 默认已经能访问）

## 4. 找 10 个人测试 · 不问"觉得怎样"

> 问"觉得怎样"→ 全是客套话。  
> 问"会用 / 愿付 / 付多少"→ 才是真话。

### 4.1 找谁（10 人结构）

| 角色 | 数量 | 优先级 |
| --- | --- | --- |
| 短剧 / 短视频 导演 / 编剧 | 4 | 最高（直接对标 AI 短剧模板） |
| 付费课程 / 知识 IP 主理人 | 3 | 高（对标创业知识库模板） |
| 展馆 / 沉浸式戏剧 / 策展人 | 2 | 中（对标记忆宫殿模板） |
| 朋友 / 同行 / 投资人 | 1 | 参考 |

### 4.2 怎么发（DM 模板 · 微信 / 私信通用）

> 我做了一周时间，把"3D 房间 + AI 讲解"做成了一个 30 秒能讲清的小工具。
>
> 不是让你买东西。是想请你花 3 分钟打开看一眼，然后告诉我 3 句真话：
>
> 1. **你会不会用？**（是 / 否 / 看情况）
> 2. **你愿不愿意为这个付钱？**（是 / 否 / 看情况）
> 3. **愿意的话，付多少？**（数字或区间）
>
> 4 个示例，挑你最想点开的看：
> - AI 短剧导演空间：https://space-memory-palace.vercel.app/share/demo-short-drama
> - AI 创业知识库：https://space-memory-palace.vercel.app/share/demo-entrepreneur
> - 个人第二大脑：https://space-memory-palace.vercel.app/share/demo-second-brain
> - 数字记忆宫殿：https://space-memory-palace.vercel.app/share/demo-memory-palace
>
> 不收你钱，也不会向你推销。哪怕"不付钱"也是答案。

### 4.3 怎么聊（5 问 · 15-20 分钟）

1. **5 秒定位**（不解释）  
   "你看 5 秒，告诉我这是干啥的。"  
   答出"3D 知识图谱 / 互动百科 / 沉浸式课程"任何一个 = 定位 OK。

2. **3 个示例的偏好**  
   "这 4 个示例（短剧 / 创业 / 第二大脑 / 记忆宫殿）哪个你最想用？"  
   → 这就是你的**主攻人群**。

3. **愿付意愿（关键题）**  
   "你愿意花多少钱让我帮你做一个这样的空间？  
   哪怕是 0 也是答案。说 0 比说'看情况'有用 100 倍。"  
   → 如果说"看情况"：  
   "那给个范围：500 以内 / 1-5 千 / 1-3 万 / 不愿意付钱？"

4. **使用场景**  
   "你会用这个东西做你的什么项目？"  
   → 出现最多的场景 = 你的客户画像。

5. **不会买的理由（关键反问）**  
   "你**不会**用这个的 3 个理由是？  
   不要客气、不要给我面子，我要听真话。"  
   → 问"不会"比问"哪里不好"信息量大 3 倍。

## 6. 方向决策（最关键的一步）

> 你可能会发现，10 个人里：
> - 4 个短剧导演觉得"还行"，但不愿付钱
> - 3 个课程老师**当场**要给你打钱
> - 2 个策展人说"我们愿意出 5 万做一次"
> - 1 个朋友说"这个方向不对"
>
> **听愿付钱的人**。其余的全是噪音。

### 真实可能性（基于你提的 5 个方向）

| 方向 | 出现的可能性 | 你的优势 |
| --- | --- | --- |
| AI 短剧导演空间 | 中 | 演示质量已经很高，但短剧行业付费意愿弱 |
| 数字展厅 / 沉浸式导览 | 高 | 一次性付 5-10 万的客户天然存在 |
| 企业知识库 | 高 | SaaS 模式，重复收入 |
| 课程空间 | 高 | 老师愿意付 1000-3000 元每月 |
| 角色世界观管理器 | 中 | 二次元 / IP 公司愿意付，但市场小 |

### 3 个立即可做的微调

1. **如果"课程空间"方向最热**：把"数字记忆宫殿"demo 改成"在线课程"demo，6 个热点变成 6 节课
2. **如果"数字展厅"方向最热**：把"AI 短剧"demo 改成"展览导览"demo，6 个热点变成 6 件展品
3. **如果"企业知识库"方向最热**：把"创业知识库"demo 改成"SaaS 培训"demo，6 个热点变成 6 个工具模块

→ **不需要重写代码**。改 `lib/seed-specs.ts` + 改 `lib/demo-spaces.ts` 的 SLUG_META 就够。

## 7. 域名绑定 runbook（10 分钟）

> 默认 `space-memory-palace.vercel.app` 已经能访问。  
> 如果你想换成自己的域名（`demo.yourbrand.com`），按下面操作。

### 7.1 在 Vercel 加域名

1. 打开 https://vercel.com/dashboard
2. Project `space-memory-palace` → Settings → Domains
3. 输入 `demo.yourbrand.com` → Add
4. Vercel 会显示一条 DNS 记录，类似：
   ```
   CNAME  demo  cname.vercel-dns.com
   ```

### 7.2 在 DNS 提供商加记录

| 提供商 | 路径 |
| --- | --- |
| 阿里云 / 腾讯云 / Cloudflare | 域名管理 → 解析设置 → 添加 CNAME 记录 |
| Namecheap / GoDaddy | Domain List → Manage → Advanced DNS |

记录值：`cname.vercel-dns.com`

### 7.3 等待 + 验证

- DNS 生效：通常 5-30 分钟（最长 48 小时）
- 验证：浏览器打开 `https://demo.yourbrand.com/share/demo-short-drama`
- HTTPS：Vercel 自动签发 Let's Encrypt 证书，**不需要**手动操作

### 7.4 短链工具（可选）

如果你要发到微信（屏蔽外链）、短信（限制字数），用下面 3 个工具之一：

- [bit.ly](https://bitly.com) - 海外常用
- [sina.lt](https://sina.lt) - 微博官方
- 自建：用你自己的域名 + 一个简单的重定向脚本

## 8. Vercel 环境变量 · 让 AI 真的"会讲"

| 变量 | 必填 | 不配会怎样 | 去哪拿 |
| --- | --- | --- | --- |
| `OPENAI_API_KEY` | 强烈建议 | AI 讲解回退到"离线示例"，带橙色角标 | https://platform.openai.com/api-keys |
| `OPENAI_MODEL` | 可选 | 默认 `gpt-4o-mini` | — |
| `NEXT_PUBLIC_SUPABASE_URL` | 可选 | 数据全在 localStorage，**换设备 = 没数据** | Supabase Project → Settings → API |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | 可选 | 同上 | 同上 |
| `NEXT_PUBLIC_CONTACT_URL` | 可选 | 分享页底部 CTA 指向默认 mailto | 你自己的联系方式 |

配完后 Vercel 会自动重新部署（30 秒）。重新打开示例 → 点"AI 讲解" → 看到的不再是"离线示例"角标，是真实讲解。

## 11. 一句话总结

> 你的产品不是"3D 知识宫殿"。  
> 你的产品是**"让一群人把一件事讲清楚的最快方式"**。
