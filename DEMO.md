## 2. 这一版改了什么

针对你的 5 个关键体验问题，逐条对应：

1. **手机端能不能看**  
   - 分享页在 < lg 屏用底部卡片 + 3D 占满上方  
   - 任何页面 3D 视图下方都有「热点 chip 横条」，不用看右侧 panel 也能切换  
   - 编辑/浏览页在窄屏 grid 自动变成上下分区（55% / 45%）

2. **3D 加载失败有没有 fallback**  
   - `SpaceViewer` 检测到 `splatStatus === "error"` 或没 `sceneUrl` 时，自动显示占位房间（地板 + 4 面墙 + 网格）  
   - 加载中显示进度条；加载失败显示琥珀色错误条 + 占位房间  
   - `SplatScene.ts` 内部已 `sharedMemoryForWorkers: false`，无需跨域隔离头

3. **热点点击准不准**  
   - 修了一个真实 bug：之前可见的圆环（半径 0.18-0.32）不可点击，只有中心小球（半径 0.13）能点  
   - 现在每个热点加了一个不可见的大球（半径 0.4）作为点击命中区，整圈光环都能点  
   - 圆环本身也加了 `userData.hotspotId`，鼠标悬停时光标会变 pointer

4. **AI 有没有胡说**  
   - 系统 prompt 明确禁止编造、强制中文、不用 Markdown 标题、不能寒暄开场  
   - 资料不足（去除标点后 < 8 字符）直接拒绝：返回"当前卡片资料不足，无法可靠讲解"  
   - 没配 `OPENAI_API_KEY` 时返回离线示例（带「离线示例」角标），不会假装是 AI 输出

5. **分享页像不像产品**  
   - 整页重建为产品落地页：顶部品牌 + 复制链接 + 主 CTA；hero 显示标题 + 一句话价值主张；3D 上方有 chip 横条；右侧/底部显示选中卡片；底部固定 CTA「咨询做一个」+ `mailto` 链接（`NEXT_PUBLIC_CONTACT_URL` 控制）

## 3. 30 秒演示视频脚本

> 录屏工具：QuickTime Player → File → New Screen Recording（Mac 自带，免费）  
> 长度：30 秒整。  
> 背景：纯黑。  
> 浏览器：Safari 或 Chrome，dev tools 关闭，URL 栏留着能看见域名。

| 秒数 | 画面 | 口播 / 字幕 |
| --- | --- | --- |
| 0-3 | 浏览器打开 `space-memory-palace.vercel.app/share/{demo-id}`，从外面滑动到 3D 房间 | 字幕：「我有一个 3D 房间。」 |
| 4-7 | 鼠标拖拽旋转 3D 房间，露出地面上 6 个彩色热点 | 「我把 AI 短剧的所有资料，挂在了这个房间里。」 |
| 8-12 | 鼠标点击「角色墙」热点 → 右侧卡片滑入，显示「林昭 / 苏蔓 / 老周 / 陈总」 | 「点击这个热点，AI 短剧的整面角色墙就出来了。」 |
| 13-17 | 点击卡片右下「AI 讲解」→ 弹窗显示 4 个风格 → 选「短讲」→ 出现讲解文本 | 「点一下，AI 就给你讲这个角色。」 |
| 18-22 | 关闭弹窗 → 切换到「分镜区」chip → 卡片换成镜头列表 | 「每一个分镜、每一条提示词，都是一个可点击的热点。」 |
| 23-27 | 点击底部「咨询做一个」按钮 → 浏览器提示打开邮件 | 「如果你的剧本、课程、展馆也想变成这样一个空间——」 |
| 28-30 | 镜头拉远，整个页面 | 口播 + 字幕：「我把一个 3D 空间，变成了可点击、可讲解的 AI 短剧导演工作台。」 |

### 录音建议

- 用 Mac 自带麦克风录前两遍试音，让节奏卡在 30 秒
- 关键句"我把一个 3D 空间，变成了可点击、可讲解的 AI 短剧导演工作台"必须出现在最后 3 秒，结尾慢慢念
- 录完用 iMovie 剪掉中间卡顿段，导出 1080p / 30fps / h.264
- 文件名：`demo-short-drama-30s.mp4`

### 10 人访谈要点

每次聊 15-20 分钟，按这个顺序问：

1. **5 秒定位题**  
   打开 demo（不解释），让他们看 5 秒。问："你觉得这是干啥的？"
   → 如果他们答"3D 知识图谱 / 互动百科 / 沉浸式课程"中任何一个，说明定位 OK

2. **最值钱的功能**  
   "你看这 6 个热点，AI 讲解、分镜、提示词、IP Bible 导出——你觉得对你来说哪个最值钱？"
   → 听他们主动说出那个功能 → 这就是你的主卖点

3. **付费意愿（关键题）**  
   "如果让你来用这个东西，**你愿意花多少钱**让我帮你做一个这样的空间？"
   → 不引导答案，让他们说数字  
   → 如果说"看情况"，追问："那给个范围，比如 500 以内 / 1-5 千 / 1-3 万？"  
   → 如果说"这个不值钱"，问："那如果加上 AI 真人讲解、可以发到微信群、可以挂你们自己的 IP 形象，值不值？"  
   → 记下原话，不修辞

4. **使用场景**  
   "你会用这个东西做你的什么项目？"
   → 出现的场景 = 你的客户画像

5. **不会买的理由**  
   "你**不会**用这个的 3 个理由是？"
   → 问"不会"比问"哪里不好"更有信息量

## 5. Vercel 环境变量 · 配好"分享页能用 AI、能上云"

> 这一步**你自己做**，需要真实的 OpenAI / Supabase key。  
> 我可以告诉你配什么、配完是什么样。

打开 Vercel Dashboard → Project `space-memory-palace` → Settings → Environment Variables。

| 变量 | 必填 | 不配会怎样 | 去哪拿 |
| --- | --- | --- | --- |
| `OPENAI_API_KEY` | 强烈建议 | AI 讲解回退到「离线示例」，带橙色角标，看得出来不是真 AI | https://platform.openai.com/api-keys |
| `OPENAI_MODEL` | 可选 | 默认 `gpt-4o-mini` | — |
| `NEXT_PUBLIC_SUPABASE_URL` | 可选 | 数据全在 localStorage，**换设备 = 没数据** | Supabase Project → Settings → API |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | 可选 | 同上 | 同上 |
| `NEXT_PUBLIC_CONTACT_URL` | 可选 | 分享页底部 CTA 指向 `mailto:hi@example.com...` | 你自己的联系方式 |

### 配完之后

1. Vercel 会自动重新部署（30 秒）
2. 重新打开分享页 → 点"AI 讲解" → 看到的不再是"离线示例"角标，是真实的讲解文本
3. 创建一个新空间 → 刷新页面 → 数据还在 → 换台设备登录 Supabase → 数据同步
4. 分享页底部"咨询做一个"按钮 → 点开是 `mailto:` / 微信 / 你的真实联系入口

### 配 OpenAI Key 的小坑

- OpenAI Key 的额度用完时，`/api/ai/explain` 会返回 `AI 讲解失败：...`，前端会显示在讲解框里，不会崩
- 系统 prompt 已经加了资料不足保护，但仍然建议给 Key 一个 usage limit（OpenAI 后台可以设）

## 6. 分享页域名

- **现在**: `https://space-memory-palace.vercel.app/share/{id}`
- **想换成自己的域名**（如 `demo.yourbrand.com`）:
  1. Vercel Project → Settings → Domains → 添加你的域名
  2. 到你的 DNS 提供商加一条 CNAME 指向 `cname.vercel-dns.com`
  3. 等 DNS 生效（通常 5-30 分钟）
  4. 任何 `/share/{id}` 链接换成新域名即可，旧链接会 308 重定向
